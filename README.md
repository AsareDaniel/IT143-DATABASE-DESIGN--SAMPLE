# IT143 – Database Design and Analysis

This repository contains SQL scripts for practicing database design and analysis as part of IT143 coursework.

## Contents

### 01_schema
- schema.sql – Contains the database schema with tables, primary keys, and relationships.

### 02_data_import
- data_import.sql – Script to insert sample data into the tables.

### 03_crud
- crud.sql – Examples of basic CRUD operations (Create, Read, Update, Delete).

### 04_joins
- joins.sql – Examples of complex JOIN queries to combine tables.

### 05_advanced_queries (optional)
- advanced_queries.sql – More advanced queries (only included if covered in class).

## How to use
1. Import schema.sql into your SQL database.  
2. Run data_import.sql to populate tables.  
3. Practice queries in crud.sql and joins.sql.
